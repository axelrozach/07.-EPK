# Domain Dictionary — Axl Lake EPK

## Entidades de Marca
| Término | Definición |
|---------|-----------|
| **Axl Lake** | Nombre artístico del DJ/Productor. Siempre escrito "Axl Lake" (no "AXL LAKE" en texto corrido) |
| **EPK** | Electronic Press Kit. Press kit digital para booking y prensa |
| **Booking** | Proceso de contratación para eventos/shows |
| **Booker** | Persona que contrata artistas para eventos |
| **Promotor** | Organizador de eventos de música electrónica |
